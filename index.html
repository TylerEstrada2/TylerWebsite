Projects

Warehouse Robotics Path Planning

Developed a robust path planning system for autonomous warehouse robots using the A* search algorithm. The system navigates complex grid-based environments, dynamically avoiding static and moving obstacles to optimize throughput.

Implemented in Python, the code features adjustable safety margins and heuristic tuning to balance path optimality with computational efficiency, simulating real-world warehouse logistics scenarios.

Formula Buckeyes EV Accumulator Cooling

Engineered the thermal management system for the team's electric vehicle accumulator. Conducted extensive Computational Fluid Dynamics (CFD) simulations in Ansys to model heat generation and airflow.

Selected and integrated high-performance fans and custom ducting, successfully maintaining battery temperatures below the thermal derate threshold under rigorous competition drive cycles.

EcoCAR CACC Algorithm Development

Spearheaded the development of a Cooperative Adaptive Cruise Control (CACC) system using C++ and ROS2. Leveraged Vehicle-to-Everything (V2X) communication to coordinate with infrastructure and other vehicles.

The algorithm, incorporating PID and Model Predictive Control (MPC), achieved a 35% improvement in energy efficiency and reduced inter-vehicle latency by 50%.

4-DOF Robotic Arm

Designed, fabricated, and programmed a custom 4-Degree-of-Freedom robotic arm. Developed a custom inverse kinematics engine in MATLAB to calculate joint angles for precise end-effector positioning.

Implemented a closed-loop control system utilizing PID, feedforward, and ROS2 to achieve sub-millimeter accuracy, validated through complex autonomous writing and manipulation tasks.
